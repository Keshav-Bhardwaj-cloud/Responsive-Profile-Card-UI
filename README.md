# Profile Card Project

A simple and clean profile card built using HTML and CSS.

## 🚀 Features
- Responsive design (works on mobile and desktop)
- Clean and modern UI
- Flexbox layout
- Hover effects
- SVG icons support
- Handles text overflow properly

## 🛠️ Tech Used
- HTML5
- CSS3 (Flexbox, Media Queries)

## 📚 What I Learned
- How to structure a webpage properly
- Using Flexbox for layout
- Making designs responsive
- Handling overflow and text wrapping
- Basic UI design principles
